# Security

Microsoft takes the security of open-source software seriously. If you believe you have found a security vulnerability in this project, report it privately to the Microsoft Security Response Center (MSRC).

- Email: secure@microsoft.com
- Portal: https://msrc.microsoft.com

Do not open public GitHub issues for suspected vulnerabilities. Include enough detail to reproduce the issue and assess impact. MSRC will review the report and coordinate any appropriate response.
