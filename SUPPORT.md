# Support

This repository is a community-supported Microsoft sample. It is not covered by Microsoft product support or Azure support plans.

Use GitHub issues for sample-specific questions, documentation defects, and reproducible bugs. For Azure product issues unrelated to this sample, use the appropriate Azure support channel.

Security vulnerabilities must be reported privately through [SECURITY.md](SECURITY.md), not through public issues.

## Disclaimer

This sample is intended for disposable, non-production environments. It deliberately injects failures and creates billable Azure resources. Review the Makefile, Bicep templates, and experiment definitions before running them in any subscription.
