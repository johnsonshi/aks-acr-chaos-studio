# Contributing

Contributions are welcome. Use GitHub issues for bugs, questions, and enhancement proposals, and submit pull requests for focused changes.

## Pull requests

- Keep changes scoped and documented.
- Do not include secrets, tenant-specific identifiers, or production data.
- Update relevant documentation when behavior or usage changes.
- Validate changes with the smallest applicable command, such as `make lint` for Bicep-only changes.

## Microsoft Contributor License Agreement

Most contributions require agreement to the Microsoft Contributor License Agreement (CLA): https://cla.opensource.microsoft.com. When a pull request is opened, the CLA bot will indicate whether action is required.

## Code of conduct

This project adheres to the [Microsoft Open Source Code of Conduct](CODE_OF_CONDUCT.md). By participating, contributors are expected to follow it.
